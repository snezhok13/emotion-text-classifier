# Emotion Text Classifier

Классификация эмоций в тексте (6 классов).

## Стек
Python, TensorFlow/Keras, Bi-LSTM

## Результат
- **Accuracy: 93.0%** на тестовой выборке
- Классы: anger, fear, joy, love, sadness, surprise

- `emotion_text_classifier.ipynb`
- `emotion_recognition_metadata.json`
